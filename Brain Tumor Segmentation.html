Problem Statement

Brain tumor segmentation using MRI scans to identify whether a tumor is present or not

Dataset

We used the BRATS-2020 dataset. The Brain Tumor Segmentation (BraTS) 2020 dataset is a collection of multimodal Magnetic Resonance Imaging (MRI) scans used for segmenting brain tumors. The full dataset is inaccessible due to being part of competitions conducted previously; however, we were able to obtain a version that has most of the data, with a few missing data entries that we can deal with.

Data contains 369 patients in total, and each contains 155 slice with 4 channels for each slice. Overall, 158 patients are tumorous, while 211 are non-tumorous

Our dataset consists of images from brain scans, as well as masked labels indicating regions of abnormal tissue. Each image has four channels. It contains 4 main files

  1. BraTS20 Training Metadata
    Contains: Slice Path, Target, Volume, Slice, and Channels
  2. Survival Info
    Contains: Age, Survival Days, and Extent of Resection
  3. Name Mapping
    Contains: 77 Low-Grade Gliomas (LGG ) Patients and 283 High Grade Gliomas (HGG) Patients
  4. Meta Data
    Contains: Slice Path, Target, Volume and Slice

State of the art

We chose the TwoPathsCNN model, which is joined at various positions, as our baseline model. This type of model has a cascading architecture, which means that two inputs are to be taken from the image and then assembled after several activation functions to have the output of one model be the input of the next one

Proposed Updates

The Following represents our updates to the State-of-the-Art model

Update #1: Added an Attention Gates to the 2 path CNN

We incorporated attention gates into the model architecture to be able to focus on the most relevant regions of the MRI images



Update #2: Implemented a U-Net Model

We implemented a U-net model on our data, since it's well known for it's strength in medical image segmentation. However, we did a little tweek to the base model where rather than doing a pixel-by-pixel segmentation, we adjusted it to work on an image level.

Update #3: Implemented an Ensemble Model between both 2-path CNN and U-Net

We implemented an ensemble model that averages the output of both the 2 path CNN and U-Net

Conclusion

After careful consideration of all aforementioned points, it is irrefutable that brain tumor segmentation is a crucial task in medical imaging, with significant implications for early diagnosis and treatment planning.

The BraTS datasets have been instrumental in advancing the field, offering multimodal MRI data that reflect real-world clinical scenarios. Many updated model innovations improved Dice scores and accuracy across different tumor regions.

While significant progress has been made, challenges remain in achieving optimal segmentation for brain tumors. We worked on TwoPathCNN and U-Net baseline models to be able to achieve a more robust model to be able to generalize the model across another dataset (BraTS, 2020) and improve its metrics. We added attention gates and used mini-batch to improve the model. By addressing these challenges, our research on deep learning models can continue to enhance brain tumor segmentation, ultimately improving patient outcomes and enabling faster, more accurate medical decisions.

For future work recommendations, exploring other models that might not need that much RAM memory and GPU. Furthermore, enhancing Two-Path CNN would be a good option, since that would increase its accuracy. Using other Ensemble models would also be a good trial to identify the presence of the tumor.
